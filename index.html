Performance Testing

A performance test can be run by clicking the button below.

Eight test cases are run. For each test case, the number of elements in the polynomial coefficient arrays increases. The values of the coefficients are randomly generated values between 0 and 99. Each test case is run for 25 iterations and the average time to perform the multiplication operation is calculated.

The results are displayed in the table below. It may take a minute or two for the testing to complete. Testing progress can be monitored in the console.
